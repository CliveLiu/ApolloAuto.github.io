int LSL_loadHSL ( const char *  libname,
char *  msgbuf,
int  msglen 
)

Tries to load a dynamically linked library with HSL routines. Also tries to load symbols for those HSL routines that are not linked into Ipopt, i.e., HAVE_... is not defined. Return a failure if the library cannot be loaded, but not if a symbol is not found.

See also
